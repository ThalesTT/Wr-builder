## **Wild Rift Build Builder – Resumo do Projeto**

**Objetivo:**
O *Wild Rift Build Builder* é uma ferramenta para jogadores de *League of Legends: Wild Rift* que permite criar, visualizar, salvar e compartilhar builds de itens para campeões. Ele facilita a experimentação de diferentes combinações de itens, mantendo a interface simples e rápida, tanto no desktop quanto em dispositivos móveis.

---

### **Funcionalidades Principais**

1. **Seleção de Itens:**

   * O usuário pode adicionar itens a slots específicos de uma build (7 slots no total).
   * Itens especiais (botas e encantamentos) são automaticamente alocados nos slots corretos.
   * Itens de suporte possuem restrição de quantidade única.

2. **Visualização da Build:**

   * Cada slot da build exibe o item selecionado.
   * O usuário pode remover itens ou trocar de slot.

3. **Filtragem e Busca:**

   * Filtragem por tipo de item (ataque, defesa, suporte, botas, encantamento, etc.).
   * Busca por nome, com suporte a nomes em português ou inglês.

4. **Persistência e Compartilhamento:**

   * Builds podem ser salvas localmente usando `localStorage`.
   * URLs podem ser geradas para compartilhar builds completas com outros jogadores.

5. **Social Card Export:**

   * Gera um *card visual* da build com:

     * Campeão central com imagem e nome.
     * Linha horizontal de todos os itens selecionados com seus nomes.
     * Botão de download para exportar o card como imagem.
   * Funciona bem em telas grandes e pequenas (scroll horizontal em dispositivos móveis).

---

### **Stack Tecnológico**

* **Frontend:** React com Hooks (`useState`, `useEffect`, `useMemo`, `useCallback`)
* **Linguagem:** TypeScript
* **Roteamento:** React Router DOM
* **Persistência:** `localStorage`
* **Componentes Principais:**

  * `Itens.tsx` → Seleção de itens e gerenciamento de build ativa
  * `MyBuilds.tsx` → Visualização e remoção de builds salvas
  * `BuildDisplay.tsx` → Renderização da build ativa
  * `ItemDisplay.tsx` → Renderização da lista de itens filtrados
  * `SocialCardModal.tsx` → Modal para exportar build como imagem
  * Hooks customizados (`useItemCatalog`, `useBuildFromUrl`, `useBuildManager`)

---

### **Diferenciais**

* Suporte total para mobile, com rolagem horizontal de itens quando necessário.
* Geração de URLs compartilháveis sem alterar a URL do navegador em tempo real.
* Interface visual clara, permitindo que qualquer jogador veja rapidamente a build de forma organizada.
